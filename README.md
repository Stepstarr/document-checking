# 项目简介
该项目旨在实现文档标准化检测，通过给定的模板自动检测文档是否符合标准，并对格式错误进行批注和自动纠错。此项目的主要目标是提高文档的一致性和专业性，减少手动检查和修正的工作量，提高工作效率。
